# 更新紀錄

採用 [Semantic Versioning](https://semver.org/lang/zh-TW/)。

## [0.4.2] - 2026-05-20

### 變更
- **登錄表改為「每堂課一份」**：先前所有課程合併在一份 docx，現在每堂課獨立成一份，符合社會局備查格式

### 介面
- 「產生證書」頁的摘要文字隨課程數動態顯示「+ N 份登錄表」
- ZIP 結構新增 `登錄表/` 子資料夾

## [0.4.1] - 2026-05-20

### 修正
- 表格最後一列底邊框在 PDF 中消失：改用 `border-collapse: separate` + 額外像素緩衝

## [0.4] - 2026-05-20

### 變更
- **全部函式庫內嵌進單一 HTML**：~1.3 MB，無外部相依、可離線運作
- 加入 `scripts/build.js` 打包腳本

## [0.3.2] - 2026-05-20

### 修正
- 修正 Mac 上 PDF 渲染只剩標題的問題：移除 `html2pdf.js` 包裝層，直接呼叫 `html2canvas` + `jsPDF`
- PDF 渲染容器從 `position: fixed` 改成 `position: absolute`，避免 Safari 排版塌縮

## [0.3.1] - 2026-05-20

### 變更
- 字型優先序改為 Windows 先：`DFKai-SB → 標楷體 → BiauKai`
- 登錄表 docx 字型改為標楷體
- 新增字型偵測，在「設定」分頁顯示偵測結果

## [0.3] - 2026-05-20

### 新增
- 真實 PDF / docx / ZIP 產生與下載
- 印章圖檔上傳與內嵌
- Google Apps Script 整合（POST 寫入試算表）
- 進度顯示

## [0.2] - 2026-05-20

### 新增
- 員工主檔加入「職稱」欄位
- Excel 匯入辨識更多欄位名稱（職稱、出生日期、生份證字號 等）
- 生日格式自動辨認（西元 / 民國）

### 變更
- 字型改用標楷體（BiauKai / DFKai-SB）
- 證書版型調整為「主任 [簽名章]」順序

## [0.1] - 2026-05-20

### 新增
- 初版原型：員工主檔、產生證書介面、預覽、模擬產生
- 民國年日期輸入
- 瀏覽器 localStorage 儲存
- Excel 匯入匯出
