# 使用手冊（給社工）

這份文件給日常實際使用工具的社工看，假設「機構負責人或 IT 已完成第一次設定」。

## 開啟工具

桌面上應該有一個 `cert-generator.html` 檔案。雙擊它，會用你預設的瀏覽器打開。

如果還沒有這個檔案，請跟機構負責人索取。

## 第一次使用前的小確認

打開工具後，建議先做一次健康檢查：

1. 點到上方「**設定**」分頁
2. 看最下面「字型偵測」應該是**綠色的「✓ 已偵測到 ⋯」**
3. 看「印章圖檔」應該已經有兩張印章圖預覽
4. 看「Google 試算表連結」應該已經填好

如果有任何一項缺，請聯絡機構負責人補上。

## 員工主檔管理

### 一次性：匯入員工名單

第一次使用時，要把員工名單匯入：

1. 到「**員工主檔**」分頁
2. 把員工資料 Excel 準備好（欄位需要：姓名、身分證字號、出生日期；職稱選填）
3. 按右上「**匯入 Excel**」，選你的檔案
4. 跳出訊息「已匯入 X 筆」就完成了

> **Excel 格式範例**見 `examples/employees-template.csv`，欄位標題只要叫「姓名」「身分證字號」「出生日期」「職稱」就會自動辨認。
>
> 生日可以是民國年（如 `80/5/12`）或西元年（如 `1991/5/12`），工具會自動辨認。

### 日常：新增離職員工

新員工進來時：

1. 「員工主檔」分頁右上「**＋ 新增員工**」
2. 填姓名、職稱、身分證、生日
3. 按「儲存」

員工離職時：

1. 在員工列表找到他，按右側「**刪除**」
2. 確認對話框按「確定」

（注意：刪除的是「員工主檔」，但歷史紀錄不會被刪。已經發過的證書 PDF 跟 Google 試算表的紀錄都會保留。）

### 定期：備份員工資料

員工主檔存在這台電腦的瀏覽器裡。**建議每月或每季匯出一次 Excel 備份**，避免電腦壞了或瀏覽器資料被清掉時資料消失。

備份方法：「員工主檔」分頁 → 右上「**匯出 Excel**」 → 存到雲端硬碟或安全的地方。

## 產生研習證明書

這是日常最常用的功能。每次有訓練結束後：

### 步驟 1：選擇參加員工

到「**產生證書**」分頁，第一個區塊是「選擇參加員工」。

- 用搜尋框找人，或直接捲動清單勾選
- 按「全選」會選取目前顯示的所有人
- 已選的人左上會顯示「**已選 N 位**」

### 步驟 2：填訓練資訊

第二個區塊，填這場訓練的基本資料：

| 欄位 | 怎麼填 |
|---|---|
| 文號 | 社會局核發的「南市社身字第 XXX 號」的 XXX 部分 |
| 辦理單位 | 預設已填好，通常不用改 |
| 社會局核發/備查日期 | 社會局公文上的核發日 |
| 訓練辦理日期 | 訓練實際進行的那天 |
| 證書簽發日期 | 證書上要印的日期，預設今天 |

所有日期都填**民國年**三碼（例：114 年 11 月 20 日 → 在「年」欄填 `114`、「月」欄填 `11`、「日」欄填 `20`）。

### 步驟 3：填課程清單

第三個區塊，把這場訓練的所有課程列出來：

- 一列填一堂課：課程名稱、講師、時數
- 課程多就按「**＋ 新增課程**」加列
- 不需要的列按右邊的 `×` 刪掉
- 下方會自動加總「總時數」

### 步驟 4：預覽 → 產生

往下滑到最底下：

1. 先按「**預覽第一張**」看版型對不對（會跳出一個彈窗顯示證書樣式）
2. 沒問題就按「**全部產生**」
3. 跳出進度條，30 人大約 30~60 秒
4. 完成後會**自動下載一個 ZIP 檔**到你的「下載」資料夾

### ZIP 檔裡有什麼

```
研習證明書_114年11月20日.zip
├── 證書/
│   ├── 林雅婷_114年11月20日_研習證明.pdf
│   ├── 陳建宏_114年11月20日_研習證明.pdf
│   └── ...（一人一份）
└── 登錄表/
    ├── 在職訓練登錄表_114年11月20日_個案研討.docx
    ├── 在職訓練登錄表_114年11月20日_職場安全.docx
    └── ...（一堂課一份）
```

- **證書 PDF**：可以直接列印，或寄電子檔給員工
- **登錄表 docx**：用 Word 開啟，可以微調後送社會局備查

### 同步寫入 Google 試算表

如果機構已設定好 Google 試算表整合，產證書時會**自動把這批紀錄寫入試算表**。完成後右下角會跳「已寫入試算表 N 筆」。

可以到「**歷史紀錄**」分頁按「前往試算表」查詢過往發放紀錄。

## 常見狀況

**Q: 我不小心刪錯員工怎麼辦？**

如果有定期匯出 Excel 備份，重新匯入即可。如果沒有，可能要手動再新增一次。

**Q: 證書文字位置跑掉了？**

可能是字型問題。到「設定」分頁看字型偵測有沒有亮綠燈。如果還是不對，截圖傳給負責人。

**Q: 同一場訓練不小心產了兩次怎麼辦？**

PDF 跟登錄表本機檔案重複下載沒影響，刪一份即可。Google 試算表上會多寫一批紀錄，可以手動把重複的列刪掉。

**Q: 換電腦了，員工主檔不見了**

員工主檔存在原本那台電腦的瀏覽器裡。如果有匯出 Excel 備份，在新電腦匯入即可。如果沒有，要重新建立。**所以提醒：定期匯出備份很重要。**

**Q: 想要修改某個員工的身分證字號**

到「員工主檔」找到他，按「編輯」修改即可。

**Q: 印章歪掉、太大、太小？**

到「設定」重新上傳新的印章圖。建議用 600×600 像素的 PNG 去背檔。

**Q: 沒有網路可以用嗎？**

可以！PDF、登錄表都會正常產出。只有「寫入 Google 試算表」這一步需要網路，沒有的話那次就不會寫入，但下次有網路時也補不回去。所以建議**有網路時操作**。

## 求救

工具壞了、產出來怪怪的、有想新增的功能 —— 請聯絡機構負責人或工具的開發者。
